# Enterprise SOC Home Lab

## Project Goal

Build an enterprise-style Security Operations Center (SOC) home lab using the Elastic Stack to simulate real-world security monitoring, detection engineering, threat hunting, and incident response.

---

## Objectives

- Learn SIEM architecture
- Deploy Elasticsearch
- Deploy Kibana
- Configure Fleet
- Configure Elastic Agent
- Deploy Active Directory
- Configure Windows Event Forwarding
- Deploy Sysmon
- Build detection rules
- Map detections to MITRE ATT&CK
- Perform threat hunting
- Investigate simulated attacks

---

## Lab Components

| System | Purpose |
|---------|----------|
| Ubuntu Server | Elasticsearch, Kibana, Fleet |
| Windows Server 2022 | Active Directory, DNS, Windows Event Collector |
| Windows 11 | Domain workstation and telemetry source |
| Kali Linux | Attack simulation |
